1. Inicio de Sesión

El módulo de inicio de sesión permite a los usuarios acceder al sistema PET-TIME mediante correo electrónico y contraseña, garantizando la seguridad de la información.

2. Registro de Usuarios

El módulo de registro permite crear una cuenta en PET-TIME para gestionar mascotas, agendar citas médicas y consultar historiales clínicos.

Durante el registro se solicitan datos como nombre, correo electrónico, teléfono y contraseña para garantizar la identificación del usuario.

4. Gestión de Citas

El módulo de citas permite programar consultas veterinarias para las mascotas, facilitando la organización de horarios y la atención médica.

Pasos para agendar una cita

  1. Seleccionar la mascota.
  2. Elegir el veterinario.
  3. Escoger fecha y hora.
  4. Confirmar la cita.
